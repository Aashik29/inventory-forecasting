# AI-Powered Inventory Purchase Forecasting System

An end-to-end machine learning pipeline built to optimize inventory purchasing decisions and minimize waste using production-oriented software architectures.

## Project Structure
- `data/`: Ingestion directory for raw datasets (gitignored).
- `src/`: Modular codebase containing preprocessing, feature engineering, modeling orchestrators, and explainability modules.
- `main.py`: Production entry point to execute the full end-to-end workflow.

## Tech Stack
Python, Pandas, Scikit-learn, XGBoost, LightGBM, CatBoost, Statsmodels, SHAP.
